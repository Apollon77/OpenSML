# OpenSML

Smart Message Language (v. 1.0.3 ) Implementation with Node.js 4.x+

!!!
 This is not a complete tested version.
 Wrong inputs could hang-up or damage your system
 Use it by your own risk
!!!

## Install

```
$ npm install open-sml --save
```

## Usage



## Developing

```
OpenSML$ npm install
OpenSML$ grunt build
OpenSML$ grunt test

```
